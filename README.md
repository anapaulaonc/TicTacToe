# TicTacToe

## Requirements

- Lua languege install
  - sudo apt install lua5.4
- Lua socket install
  - sudo apt-get install lua-socket
- Love2d Install
  - sudo apt-get install love

## Start Server

inside TicTacToe folder run:
`lua server.lua`

## Start Client

inside TicTacToe folder run:
`love .`
